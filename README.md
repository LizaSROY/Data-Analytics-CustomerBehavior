## Project Steps

1. **Data Loading**

   * Imported raw dataset into Python using Pandas
   * Checked data types and structure

2. **Exploratory Data Analysis (EDA)**

   * Summary statistics
   * Distribution analysis
   * Trend and correlation analysis
   * Identification of missing values and outliers

3. **Data Cleaning**

   * Handled missing values
   * Removed duplicates
   * Corrected data types
   * Standardized column names

4. **SQL Analysis (MySQL)**

   * Loaded cleaned data into MySQL
   * Wrote SQL queries to:

     * Aggregate metrics
     * Filter and segment data
     * Identify trends and top-performing categories

5. **Data Visualization**

   * Connected MySQL data to Tableau
   * Built an interactive dashboard with key KPIs and charts

6. **Reporting & Presentation**

   * Created a written analysis report
   * Designed a professional presentation using Canva to communicate insights
